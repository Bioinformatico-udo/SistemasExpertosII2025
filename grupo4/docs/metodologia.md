# Metodología KommonCats

KommonCats es un marco para estandarizar caracteres morfológicos y ecológicos con el fin de:
- Reducir ambigüedad entre observadores.
- Generar datos estructurados para claves dicotómicas y sistemas expertos.
- Facilitar la integración con modelos interpretables.

## Pasos principales
1. Definición y estandarización de caracteres.
2. Recolección y registro con metadatos.
3. Codificación y cálculo de features.
4. Validación experta y pruebas inter-observador.
5. Integración en motor de reglas y modelo ML.
6. Evaluación y mejora continua.
