# SEITH - Sistema Experto de Identificación 🦀🚀

Bienvenido al **Sistema Experto de Identificación de Taxones de Hippidae**. Esta herramienta utiliza Visión Artificial e Inteligencia Artificial para asistir en la clasificación taxonómica de los cangrejos topo.

## 📋 Requisitos Previos
*   Python 3.10+
*   Acceso a Internet (Para la consulta con el experto IA)

## 🛠️ Instalación Rápida
1.  **Clonar/Descargar** el proyecto en tu máquina.
2.  **Crear Entorno Virtual**:
    ```cmd
    python -m venv venv
    ```
3.  **Activar Entorno**:
    *   Windows: `venv\Scripts\activate`
    *   Linux/Mac: `source venv/bin/activate`
4.  **Instalar Dependencias**:
    ```cmd
    pip install -r requirments.txt
    ```

## 🚀 Ejecución
Para iniciar el sistema experto:
```cmd
python main.py
```

## 👩‍🏫 Credenciales de Acceso
*   **Modo Admin**: usuario: `admin` | clave: `admin123`
*   **Modo Alumno**: usuario: `invitado` | clave: `user123`
*(O puedes registrar tu propio usuario en la pantalla de inicio)*

## 🧠 Configuración de la IA
Para interactuar con el profesor experto necesitas una API Key de Groq (Gratuita):
1.  Visita [Groq Cloud Console](https://console.groq.com/keys).
2.  Genera una clave y pégala dentro de la aplicación en el campo de la IA.
3.  El sistema la recordará automáticamente.
